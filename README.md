# NeoWarp

一款基于TurboWarp的Scratch编辑器，专为中文用户优化。

## 特性

- 🎨 **毛玻璃效果**：创建变量、列表、自定义积木时的背景采用毛玻璃效果
- 📊 **系统监控**：顶栏实时显示CPU和RAM占用率
- 🔤 **鸿蒙字体**：使用HarmonyOS Sans字体，界面更加美观
- 🖼️ **分离舞台**：支持将舞台分离到独立窗口显示
- 🎭 **过渡动画**：舞台全屏/退出全屏时带有流畅的过渡动画
- 🔲 **圆角设计**：多处UI采用圆角设计，更加现代

## 与原版的不同

- 在一些地方加入了毛玻璃效果
- 加入了过渡动画
- 加入了圆角
- 加入了内存和CPU占用率监控
- 支持分离舞台功能
